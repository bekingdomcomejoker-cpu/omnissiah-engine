- Node.js 22.13.0+

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/omnissiah-engine.git
   cd omnissiah-engine
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

   ```bash
   pnpm dev
   ```

   Open your browser and navigate to `http://localhost:3000`
